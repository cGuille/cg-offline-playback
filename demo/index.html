Standalone track sample

Code


<cgop-track
  key="first-track"
  label="First track displayed label"
  confirm-download-tpl="#confirm-download"
  url="/demo/sample.mp3">
</cgop-track>

<template id="confirm-download">
  <div>
    <p>
      Do you want to start downloading "<span class="track-label"></span>" now?<br />
      You may want to be connected to a Wi-Fi network in order to do this.
    </p>
    <button class="confirm">Download</button>
    <button class="cancel">I'll do it later</button>
  </div>
</template>

Playlist code sample

If the track is in a playlist, only one of them is played at a time.

Code


<cgop-playlist confirm-download-tpl="#confirm-download">
  <cgop-track
    key="first-track"
    label="First track displayed label"
    url="/demo/sample.mp3">
  </cgop-track>

  <cgop-track
    key="other-track"
    label="Other track"
    url="/demo/sample.mp3">
  </cgop-track>
</cgop-playlist>

<template id="confirm-download">
  <div>
    <p>
      Do you want to start downloading "<span class="track-label"></span>" now?<br />
      You may want to be connected to a Wi-Fi network in order to do this.
    </p>
    <button class="confirm">Download</button>
    <button class="cancel">I'll do it later</button>
  </div>
</template>
