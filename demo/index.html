Playlist code sample

If the track is in a playlist, only one of them is played at a time.

Code


<cgop-playlist>
  <cgop-track
    key="first-track"
    label="First track displayed label"
    url="/demo/sample.mp3">
  </cgop-track>

  <cgop-track
    key="other-track"
    label="Other track"
    url="/demo/sample.mp3">
  </cgop-track>
</cgop-playlist>

Result
